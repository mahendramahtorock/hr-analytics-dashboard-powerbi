 # HR Analytics Dashboard (Power BI)

## Overview

This project is a Human Resource Analytics Dashboard created using Power BI.
It helps to understand employee data, track attrition, and analyze workforce trends in a simple and visual way.

The dashboard is useful for HR teams and business analysts to make better decisions based on employee data.


## Problem Statement

Many companies face problems like:

* High employee attrition
* Low job satisfaction
* Lack of clear workforce insights

This dashboard solves these problems by:

* Showing attrition trends
* Analyzing employee demographics
* Highlighting key areas where improvements are needed


## Key Features

### 1. Employee Overview

* Total Employees: 1470
* Active Employees: 37
* Attrition Count: 237
* Attrition Rate: 16.12%


### 2. Department-wise Attrition

* Visual breakdown of attrition in:

  * R&D
  * Sales
  * HR
* Helps identify which department has the highest attrition


### 3. Age Group Analysis

* Employees grouped by age:

  * Under 25
  * 25–34
  * 35–44
  * 45–54
  * Above 55
* Gender comparison included


### 4. Job Satisfaction Rating

* Ratings from 1 to 4 for different roles
* Helps understand employee satisfaction across job roles


### 5. Education Field Analysis

* Attrition based on education background:

  * Life Sciences
  * Medical
  * Marketing
  * Technical
  * HR
* Helps identify trends in employee exits


### 6. Attrition by Gender and Age Group

* Detailed breakdown of attrition across:

  * Gender
  * Age groups
* Helps in targeted HR strategies


## Tools Used

* Power BI
* Data Visualization
* Data Cleaning and Transformation


## Insights from Dashboard

* Highest attrition is seen in certain departments like Sales
* Employees in the 25–34 age group show higher attrition
* Job satisfaction varies significantly across roles
* Education background also impacts attrition trends


## How to Use

1. Download the Power BI file (.pbix)
2. Open it using Power BI Desktop
3. Explore different visuals and filters
4. Use insights for business decision making
